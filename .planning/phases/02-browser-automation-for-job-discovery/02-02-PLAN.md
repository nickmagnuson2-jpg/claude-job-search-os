**`score_role(role: dict, context: dict) -> int`:**
Per D-04, four dimensions with weights:

1. **Title match (weight=0.35):** `max(SequenceMatcher(None, role["title"].lower(), t.lower()).ratio() for t in context["target_titles"])` * 10. If no target_titles, score 5.

2. **Seniority match (weight=0.25):** Extract seniority from role["title"] and from context["target_seniority"]. Score = 10 - abs(role_level - target_level) * 2, clamped to 1-10. If no target seniority, score 5.

3. **Industry/domain match (weight=0.20):** Check if role["department"] or role["team"] fuzzy-matches any context["target_industries"] (SequenceMatcher ratio > 0.6). Score 8 if match, 3 if no match. If no target_industries, score 5.

4. **Keyword overlap (weight=0.20):** Count how many context["skills"] appear (case-insensitive substring) in role["description_plain"]. Score = min(10, matched_count * 2). If no skills or no description, score 3.

Final: `round(0.35*d1 + 0.25*d2 + 0.20*d3 + 0.20*d4)`, clamped 1-10.

Per D-05: All roles get scores, no filtering threshold.

**`is_duplicate(role_title: str, role_company: str, pipeline_entries: list[dict]) -> bool`:**
Per D-10: Exact company match (case-insensitive) + fuzzy title match (SequenceMatcher >= 0.80).

```python
from difflib import SequenceMatcher

def is_duplicate(role_title: str, role_company: str, pipeline_entries: list[dict]) -> bool:
    company_lower = role_company.lower().strip()
    for entry in pipeline_entries:
        if entry["company"].lower().strip() != company_lower:
            continue
        ratio = SequenceMatcher(None, role_title.lower(), entry["role"].lower()).ratio()
        if ratio >= 0.80:
            return True
    return False
```
